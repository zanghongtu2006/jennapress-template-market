Ένα Jenna Press site πρέπει να μπορεί να δημοσιευθεί ως static output και να δοκιμαστεί ως static output. Αυτό σημαίνει ότι οι έλεγχοι deployment πρέπει να εστιάζουν στο παραγόμενο HTML, στους εσωτερικούς συνδέσμους, στα language paths και στο browser persistence για theme και γλώσσα.

Η συντήρηση πρέπει να παραμένει εξίσου πειθαρχημένη. Ενημερώστε πρώτα το περιεχόμενο σε markdown. Αναδημιουργήστε έπειτα όποια παράγωγα αρχεία χρειάζονται. Ελέγχετε τις αλλαγές πλαισίου χωριστά από τις αλλαγές περιεχομένου όποτε είναι δυνατό.

Ένα τέτοιο έργο παραμένει υγιές όταν η δημοσίευση μένει τόσο απλή ώστε η ομάδα να μπορεί να την εξηγήσει σε μία παράγραφο.
