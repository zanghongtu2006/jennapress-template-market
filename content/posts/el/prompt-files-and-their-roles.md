Το Jenna Press διατηρεί δύο prompt files επειδή η δημιουργία template και η συντήρηση περιεχομένου είναι σχετικές, αλλά όχι ταυτόσημες, εργασίες. Ένα template prompt πρέπει να βοηθά ένα AI να παράγει output συμβατό με το framework. Ένα maintenance prompt πρέπει να βοηθά ένα AI να ενημερώνει το περιεχόμενο χωρίς να σπάει επιπόλαια τη δομή.

Ο διαχωρισμός αυτών των ευθυνών κρατά τις οδηγίες πιο ακριβείς. Κάνει επίσης ευκολότερο τον έλεγχο μεταγενέστερων αλλαγών, επειδή η πρόθεση κάθε prompt file παραμένει ορατή.

Ένα prompt file είναι επομένως εργαλείο συνεργασίας. Πρέπει να κωδικοποιεί όρια, αναμενόμενη δομή output και πειθαρχία review.
