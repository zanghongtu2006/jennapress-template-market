Jenna Press keeps two prompt files because template generation and content maintenance are related, but not identical, jobs. A template prompt should help an AI produce framework-compatible output. A maintenance prompt should help an AI update content without casually breaking structure.

Treating these as separate prompt responsibilities keeps instructions more precise. It also makes it easier to review later edits, because the intent of each prompt file remains visible.

A prompt file is therefore a collaboration tool. It should encode boundaries, expected output structure, and review discipline.
